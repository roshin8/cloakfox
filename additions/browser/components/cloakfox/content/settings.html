How Cloakfox works

Cloakfox is a Firefox build, so it always claims to be Firefox. Per-container privacy comes from which Firefox user you appear to be — Mac, Windows, or Linux Firefox with varied hardware, screen, GPU, fonts, and audio characteristics drawn from BrowserForge's real-world joint distribution.

Each container's master seed deterministically samples one fingerprint at first launch. The sample is coherent — webGl renderer matches platform matches screen — so individual fields don't contradict each other. Click Sample a new persona to roll a new master seed and draw a new Firefox identity. Click any field below to override that one value while keeping the rest of the persona intact.

We can't credibly spoof as Chrome / Safari / Edge because the JS engine (SpiderMonkey), CSS engine (Gecko), TLS fingerprint, and WebRTC stack are all Firefox-specific and visible to sophisticated probes. A shallow Chrome UA disguise (for sites that only check the UA string) is a future opt-in feature.
